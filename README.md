# 🤖 AI SOP Assistant

An AI-powered chatbot that answers questions based on SOP documents using Retrieval-Augmented Generation (RAG) and semantic search.

---

## 🚀 Features

* Chat-based interface for user interaction
* SOP-based question answering
* Semantic search using TF-IDF and cosine similarity
* Intent handling (summary, explanation queries)
* Controlled responses to avoid hallucination

---

## 🧠 How It Works

1. User asks a question
2. System performs semantic search on SOP data
3. Most relevant chunk is retrieved
4. AI generates a concise answer based only on SOP

---

## 🛠️ Tech Stack

* Python
* OpenAI API
* Streamlit
* scikit-learn

---

## 📂 Project Structure

* `day4_rag_advanced.py` → RAG implementation
* `app.py` → Web chatbot (Streamlit)
* `day6_semantic_rag.py` → Semantic search version
* `sop.txt` → Sample SOP data

---

## 💡 Key Learning

* Transitioned from keyword-based search → semantic search
* Improved accuracy using TF-IDF + cosine similarity
* Applied prompt engineering for controlled outputs
